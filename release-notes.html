Release Notes

1.0.2.77-allspice 2025-04-09

General

  • Each release is now given a "Spice" nickname, for easier recognition.
    • Allspice is our first new release!

Text Expansion Logic

  • Adaptive Casing now checks only the first character, for simplicity.
    • See 'Adaptive Casing' under Working With Case Sensitivity for details.
  • When multiple Instant Text Replace items trigger at the same time, they are now shown in the Pop-up Preview.
    • Before, whichever is the first one would get expanded instantly.
  • Chaining no longer breaks when you expand a link that contains a newline.
  • Text Replace expansions that don't actually alter the content (e.g., replacing 'abc' with 'abc') no longer get triggered.

Pop-up Preview

  • Snippet Preview Text and Labels are now limited to 30 characters, to prevent the Pop-up Preview from getting too big.
  • Typing history is now cleared when CTRL+A is typed.
  • Snippet Preview now shows Whitespace (consecutive spaces, tabs and newlines) in a compact manner.

Options (Bottom cog)

  • A new Preferences Dialog is available in the Settings Cog.
    • [New Snippets] Here you can set the Mode, Case Sensitivity and Trigger Settings of New Snippets.
    • [Desktop] The Run PepperText on Windows startup option has moved here.

Installer

  • Fixed rare startup crash when run from deep file paths.

1.0.1.20-beta 2024-05-24

General

  • Added protection against simultaneous PepperText instances.
    • After launching PepperText, subsequent launch attempts will just cause the first instance to be brought to view.
  • Fixed tooltips to be more prevalent, and a lot more responsive.
  • Fixed the issue where drag-drop can't drop if hovered over a textbox.
  • (Mostly) Fixed the issue where on startup, text expansion hasn't been suspended (meaning pop-up/text expansion is triggered on the interface itself).

Navigation Pane (Left side)

  • Added an easier-to-discover 'Add' Folder/Page button, when you hover over the Snippet Roots/Folder.
  • Added a confirmation pop-up before a Folder/Page gets deleted, to avoid accidental deletion.
  • Added the automatic creation of a new Page, within creating a new Folder.
  • Changed the Snippets Root from a hard drive to a binder.
  • Changed it so that the Snippets Root and Folders can no longer be activated/deactivated. This can only be done for Pages now, for simplicity.

Main Window (Right side)

  • Added 'Learn more here' website links, for Label and Shortcut Settings.
  • Changed the Context option:
    • Its name is now Shortcut Settings.
    • Its location is now under the new Advanced Options (cog icon on the right of each Snippet).
    • It now has 3 simple options instead of 5 specific ones.
  • Changed the name of the Text Replace (Instant) mode to Instant Text Replace.
  • Changed the starting snippets (for brand new installs) to just a single Page of simple examples.

Options (Bottom cog)

  • Added Contact us.
    • Opens the Contact Us page of the PepperText website.
  • Added Auto-start PepperText On Windows Startup.
    • If enabled, when you log into Windows, PepperText will automtically startup and run in the background. As usual, clicking on the System Tray brings back the interface.
  • Added Exit.
    • Immediately quits the application.

About Box

  • Added links to peppertext.com and Privacy Policy.
  • Added EULA and Credits.
  • Added fun animation when clicking on the logo.
  • Changed the design.

Search (Top-left corner)

  • Fixed the issue where clicking on a search result sometimes doesn't scrolls it into view.

Pop-up

  • Fixed the issue where the pop-up can obscured behind the taskbar.

Installer

  • Added Start automatically on login option.
  • Installer is now signed (no more warnings on download and launch).
    • If enabled, when you log into Windows, PepperText will automtically startup and run in the background. As usual, clicking on the System Tray brings back the interface.
