Release Notes

1.0.1.xx 2024-04-13

General

  • Added protection against simultaneous PepperText instances.
    • After launching PepperText, subsequent launch attempts will just cause the first instance to be brought to view.
  • Fixed tooltips to be more prevalent, and a lot more responsive.
  • Fixed the issue where drag-drop can't drop if hovered over a textbox.
  • Fixed the issue where on startup, text expansion hasn't been suspended (meaning pop-up/text expansion is triggered on the interface itself).

Navigation Pane (Left side)

  • Added an easier-to-discover 'Add' Folder/Page button, when you hover over the Snippet Roots/Folder.
  • Added a confirmation pop-up before a Folder/Page gets deleted, to avoid accidental deletion.
  • Added the automatic creation of a new Page, within creating a new Folder.
  • Changed the Snippets Root from a hard drive to a binder, which better fits the folder+page theme.
  • Changed it so that the Snippets Root and Folders can no longer be activated/deactivated. This can only be done for Pages now, for simplicity.

Main Window (Right side)

  • Changed the Context option to exist inside Advanced Settings instead, and the Context item-keyword is now known as Expansion Control.
  • Changed the Context (now known as Expansion Control) to have 3 simple options instead of 5 specific ones.
  • Changed the Text Replace (Instant) mode to be instead named as Instant Text Replace.

Options (Bottom cog)

  • Added Contact us.
    • Opens the Contact Us page of the PepperText website.
  • Added Auto-start PepperText On Windows Startup.
    • If enabled, when you log into Windows, PepperText will automtically startup and run in the background. As usual, clicking on the System Tray brings back the interface.
  • Added Exit.
    • Immediately quits the application.

About Box

  • Added links to peppertext.com and Privacy Policy.
  • Added EULA and Credits.
  • Added fun animation when clicking on the logo.
  • Changed the design.

Search

  • Fixed the issue where clicking on a search result sometimes doesn't scrolls it into view.

Pop-up

  • Fixed the issue where the pop-up can obscured behind the taskbar.

Installer

  • Added Select Destination Location page, to allow a custom install location.
  • Added Start automatically on login option.
    • If enabled, when you log into Windows, PepperText will automtically startup and run in the background. As usual, clicking on the System Tray brings back the interface.
